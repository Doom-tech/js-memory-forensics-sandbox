Detailed Description

RAII-обёртка для глобальной инициализации V8 в процессе.

Объект нужно создать до создания экземпляров JsSandbox и держать живым до тех пор, пока все песочницы не будут уничтожены. Так жизненный цикл V8 остаётся явным и не размазывается по коду.
