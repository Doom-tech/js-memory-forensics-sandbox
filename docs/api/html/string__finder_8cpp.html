◆ locateTrackedValues()

std::vector< TrackedLocation > locateTrackedValues ( const std::vector< DumpedRegion > & regions,
const std::vector< TrackedValue > & trackedValues,

Ищет в raw dump значения, закреплённые через mark(name, value).

Значение ищется в двух вариантах: как ASCII-байты и как UTF-16LE. Это помогает находить строки, которые V8 хранит в разных внутренних представлениях.

Parameters
regionsМетаданные dump-файлов и их исходные адресные диапазоны.
trackedValuesЗначения, собранные JavaScript-песочницей.
maxHitsPerValueМаксимальное количество адресов для одного значения.
Returns
Список мест, где были найдены bytes preview отмеченных значений.
Exceptions
std::runtime_errorесли dump-файл нельзя открыть.
